# Credit-Card-Fraud-Detection-Dashboard-Power-BI
Created an interactive Power BI dashboard analyzing ₹3.31M fraudulent transactions with a 28.6% fraud rate and 10.7% critical risk. Identified “Card Not Present” as the top fraud type and visualized fraud risk levels (Low 42.4%, Medium 27.9%, High 18.8%, Critical 10.8%) across states and months for improved risk monitoring.
